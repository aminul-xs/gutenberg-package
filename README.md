# gutenberg-package
``` 
    create a new instance of the react plugin . output the result to the lib folder and file index.js
    your - gutenberg-package /
    ├── lib /
    │   ├── component /
    │   │   ├── Border.js (generated after build)
    │   │   └── Color.js (generated after build)
    │   └── index.js(generated after build)
    ├── src /
    │   ├── component /
    │   │   ├── Border.js
    │   │   └── Color.js
    │   └── index.js
    ├── .babelrc
    ├── package.json
    └── webpack.config.js
```
